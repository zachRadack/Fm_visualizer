Finite machine | Search Visualizer

by Zachary Radack
This is a step by step finite machine that visualizes search Algorithms

Instructions:

  1. If you have a seed for the RNG, put it in "RNG Seed". This will impact nodes and connections that load. Leave it black for a randomized seed.
  2. Hit "Load Graph"
  3. Select your "start node" and "end node"
  4. Select your desired "Algorithm"
  5. Set the desired total amount of nodes (N) that you want
  6. Set the desired average amount of connects (C) that you want
  7. Hit "Start"
  8. Hit "Next Step" until it finds the location
